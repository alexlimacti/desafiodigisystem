# Desafio Digisystem

## Criar uma API (Tema livre)

- Spring Boot
- Spring Cloud Feign
- Banco em memória H2
- Documentação OpenAPI



## URL's

Swagger - localhost:8080/swagger-ui/index.html

Verificação de CEP - localhost:8080/api/v1/cep/{cep}

Pessoa Post - localhost:8080/api/v1/pessoas

Pessoa Put - localhost:8080/api/v1/pessoas/{pessoaID}

Pessoa Get ALL - localhost:8080/api/v1/pessoas

Pessoa Get ID - localhost:8080/api/v1/pessoas/{pessoaID}

Pessoa Delete - localhost:8080/api/v1/pessoas/{pessoaID}
